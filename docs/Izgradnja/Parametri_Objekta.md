# Parametri Objekta

## Osnovni parametri objekta

| Parametar                  | Vrednost                                           |
|:---------------------------|:---------------------------------------------------|
| Tip objekta                | Manja stambena zgrada (novogradnja)                |
| Namena                     | Kolektivno stanovanje                              |
| Neto stambena površina     | oko **500 m²**                                     |
| Bruto građevinska površina | približno **600 – 650 m²**                         |
| Odnos bruto / neto         | približno **1.2 – 1.3**                            |
| Spratnost                  | **P + 1 + PK** (prizemlje, sprat, potkrovlje)      |
| Podrum                     | nema                                               |
| Tip krova                  | ravan krov                                         |
| Tip konstrukcije           | armirano-betonska konstrukcija sa zidanim ispunama |
| Tip gradnje                | cigla + termoizolacija                             |
| Broj stambenih jedinica    | okvirno **8 – 12 stanova**                         |
| Parking                    | spoljašnja parking mesta na parceli                |
| Lokacija projekta          | Srbija (urbanizovana zona)                         |

# Parametri parcele

| Parametar                 | Vrednost                                 |
|:--------------------------|:-----------------------------------------|
| Tip sticanja placa        | kompenzacija za stanove                  |
| Model kompenzacije        | deo stambenih kvadrata vlasniku placa    |
| Namena parcele            | stambena gradnja                         |
| Tip gradnje               | slobodnostojeći ili poluotvoreni objekat |
| Potrebna veličina parcele | približno **500 – 800 m²**               |
| Koeficijent izgrađenosti  | približno **0.9 – 1.2**                  |
| Koeficijent zauzetosti    | približno **30 – 40 %**                  |

# Urbanistički parametri objekta

| Parametar                 | Vrednost                                                        |
|:--------------------------|:----------------------------------------------------------------|
| Maksimalna visina objekta | oko **10 – 12 m**                                               |
| Broj etaža                | **3 nadzemne etaže**                                            |
| Tip etaža                 | prizemlje + sprat + potkrovlje                                  |
| Zajedničke površine       | hodnici, stepenište, ulazni hol                                 |
