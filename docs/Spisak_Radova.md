# Radovi Izgradnja

<br/>


# Detaljan spisak radova za izgradnju stambene zgrade (~500 m² neto)

Objekat:
- manja stambena zgrada
- spratnost: **P + 1 + PK**
- oko **500 m² neto stambene površine**
- **bez podruma**
- **ravan krov**
- spoljašnji parking

Napomena:
- faza **kupovine placa nije uključena**
